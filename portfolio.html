Lightbot Reflection

Lightbot is a fun challenging game that makes you program a robot in a way where it is like a puzzle or a problem. This game is like a computer in a way because it has command keys such as jump, move forward, or rotate. In a computer the command keys represent the inputs of being put into a computer like a mouse, a hard drive, or a keyboard. Next, the responses that the robot make on the game after you've selected a command key will make the robot do it as a result. For example if you click move forward, you will see that the robot will move forward. In a computer this is called an output. When you use a mouse, you can see that the cursor on the screen is moving in response to your hand's motion on the mouse. However, the difference between Lightbot and a computer is that Lightbot is just a simple programming website where basic commands are given to the robot, whereas a computer on the other hand has a lot of commands that makes it possible to do a lot of things. A computer is a more complex version of Lightbot.
